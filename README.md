# F1 Qualifying Analyses

## Live site → <https://rkk99.github.io/f1-quali/>

Single-page, fully offline qualifying-analysis dashboards for the current F1 season.

## How to use

1. Open the **[live site](https://rkk99.github.io/f1-quali/)**.
2. Pick a Grand Prix card (newest race first) to open its Q3 analysis.
3. On a race page: the default view shows the **top 5**; use the **Compare** dropdown
   to pick any two drivers for a head-to-head. Hover the dominance map, and pinch or
   scroll to zoom the telemetry charts. Use **"← All analyses"** to return here.

## What's inside

- One self-contained page per Grand Prix (no server, no CDN) — open it and it works.
- **Q3, top 10**: track-dominance map, speed / throttle / time-delta telemetry, plus
  sector and gap-to-pole charts.
- A landing page listing every race that has run this season, newest first.
